# 📊 Sales Dashboard

An interactive **Excel Sales Dashboard** built to analyze and visualize sales data across countries, product lines, and time periods.

![Sales Dashboard](dashboard-preview.png)

---

## 📌 Overview

This dashboard provides a comprehensive view of sales performance using pivot charts, slicers, and dynamic visuals — all built in Microsoft Excel. It enables quick filtering and analysis by country, product line, and year.

---

## 🔍 Features

- **Product Wise Sales** — Bar and horizontal bar charts comparing sales across Classic Cars, Motorcycles, Trucks & Buses, and Vintage Cars
- **Quarter Wise Sales** — Line chart tracking sales trends across Q1–Q4 for years 2003, 2004, and 2005
- **Country Wise Sales** — Pie and donut charts showing sales distribution by country (USA, France, Australia, Spain, etc.)
- **Total Sales Trend** — Line chart showing monthly sales trends across all years
- **Order Number vs Sales** — Dual-axis bar chart comparing Sum of Sales and Sum of Order Numbers by year and quarter
- **Interactive Slicers** — Filter data dynamically by **Country** and **Year**

## 📂 Dataset Details

- **Years Covered:** 2003, 2004, 2005
- **Countries:** USA, France, Germany, Australia, Ireland, Italy, Japan, Norway, Philippines, and more
- **Product Lines:** Classic Cars, Motorcycles, Trucks and Buses, Vintage Cars
- **Key Metrics:** Sum of Sales, Sum of Order Numbers

---

## 🚀 How to Use

1. Download the `.xlsx` file from this repository
2. Open it in **Microsoft Excel** (2016 or later recommended)
3. Use the **Country** and **Year** slicers on the left to filter the dashboard
4. All charts update automatically based on your selection
